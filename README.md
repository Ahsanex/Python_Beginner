# Python Beginner Projects

This folder contains simple Python programs for learning basics such as input/output, arithmetic, strings, functions, arrays, and file handling.

## Program Summary

- hello.py: Greets the user by name.
- area_rectangle.py: Takes the length and breadth of a rectangle, calculates the area, and prints the result.
- string.py: Takes a name as input and prints it in lowercase, along with repeated output.
- fn_test.py: Defines a function that adds two numbers and prints the sum.
- array.py: Uses NumPy to store 5 integers and prints each one with a numbered label.
- passfailtester.py: Takes passing marks and student marks, then shows whether each student passed or failed.
- test.py: Takes a number of values from the user, stores them in an array, and prints the array.
- project :
  main.py: Lets the user save and retrieve information from a JSON file.
  data.JSON: Lets Store that info In Dictionary Way
